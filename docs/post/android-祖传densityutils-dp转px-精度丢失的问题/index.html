Android 祖传DensityUtils dp转px 精度丢失的问题

2022-03-14

文章目录

祖传DensityUtils的代码

1
2
3
4
5
6

/**
 * dp转px
 */
public static int dp2px(Context context, float dpVal) {
    return (int) TypedValue.applyDimension(TypedValue.COMPLEX_UNIT_DIP, dpVal, context.getResources().getDisplayMetrics());
}

DensityUtils.dp2px的返回值px数值，因为方法内部进行了float转int，会直接去掉小数，但不会四舍五入，所以进度会丢失。比如以下数据

1
2
3
4
5
6

DensityUtils.dp2px 1dp = 2px
DensityUtils.dp2px 2dp = 5px
DensityUtils.dp2px 3dp = 8px
TypedValue.applyDimension(TypedValue.COMPLEX_UNIT_DIP) 1dp = 2.750000px
TypedValue.applyDimension(TypedValue.COMPLEX_UNIT_DIP) 2dp = 5.500000px
TypedValue.applyDimension(TypedValue.COMPLEX_UNIT_DIP) 3dp = 8.250000px

可以把TypedValue.applyDimension的float返回值，再用Math.round进行float转int四舍五入

String format = "DensityUtils.dp2px %ddp = %dpx";
int dp = 1;
System.out.printf(Locale.getDefault(), (format) + "%n", dp, DensityUtils.dp2px(MainActivity.this, dp));
dp++;
System.out.printf(Locale.getDefault(), (format) + "%n", dp, DensityUtils.dp2px(MainActivity.this, dp));
dp++;
System.out.printf(Locale.getDefault(), (format) + "%n", dp, DensityUtils.dp2px(MainActivity.this, dp));

format = "TypedValue.applyDimension(TypedValue.COMPLEX_UNIT_DIP) %ddp = %fpx";
dp = 1;
System.out.printf(Locale.getDefault(), (format) + "%n", dp, TypedValue.applyDimension(TypedValue.COMPLEX_UNIT_DIP, dp, getResources().getDisplayMetrics()));
dp++;
System.out.printf(Locale.getDefault(), (format) + "%n", dp, TypedValue.applyDimension(TypedValue.COMPLEX_UNIT_DIP, dp, getResources().getDisplayMetrics()));
dp++;
System.out.printf(Locale.getDefault(), (format) + "%n", dp, TypedValue.applyDimension(TypedValue.COMPLEX_UNIT_DIP, dp, getResources().getDisplayMetrics()));

format = "Math.round(TypedValue.applyDimension(TypedValue.COMPLEX_UNIT_DIP)) %ddp = %dpx";
dp = 1;
System.out.printf(Locale.getDefault(), (format) + "%n", dp, Math.round(TypedValue.applyDimension(TypedValue.COMPLEX_UNIT_DIP, dp, getResources().getDisplayMetrics())));
dp++;
System.out.printf(Locale.getDefault(), (format) + "%n", dp, Math.round(TypedValue.applyDimension(TypedValue.COMPLEX_UNIT_DIP, dp, getResources().getDisplayMetrics())));
dp++;
System.out.printf(Locale.getDefault(), (format) + "%n", dp, Math.round(TypedValue.applyDimension(TypedValue.COMPLEX_UNIT_DIP, dp, getResources().getDisplayMetrics())));
